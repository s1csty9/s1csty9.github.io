FAQ

Is this hard?

I made the map to be rather difficult, but I intentionally didn't make it too hard so most players should be able to beat it quickly with some effort. If you are having trouble, just keep looking around.


It's too dark! Can you fix this?

I intended for the darkness to be a mechanic in the map so that players would utilise management of the limited light sources they are given. I built it based off of the default brightness in Minecraft, and I carefully positioned external light sources to dimly light up key areas. If it's too dark still, try turning up your game brightness. You can use mods that default brightness to max, but they will effectively skip you through parts of the map you were supposed to figure out.


What versions of Minecraft can I play the map on?

I made the map on 1.7.2, so you can play it on 1.7.2 or any version released after up to the latest release.
